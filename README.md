# AccelerationGate
Đồ án môn học IE221
the first pygame project

Game được tạo full bằng Pygame
Player dùng chuột:
 - chuột phải: di chuyển
 - chuột trái: tấn công
 - chọn skill: chuột phải
Player dùng bàn phím
- di chuyển: AWSD hoặc phím điều hướng
- tấn công: phím space
- chọn skill: phím 1 2 3
